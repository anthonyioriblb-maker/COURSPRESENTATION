Chapitre 9 : Les opérations

I. Calculs posés

1) Poser une addition

Règle : Pour poser une addition, on aligne les virgules, les unités, les dizaines, les centaines... On peut ajouter des 0 si nécessaire.
Exemple : Calculer 12,4 + 3,859 + 567,7

Règle : Pour poser une soustraction, on aligne également les virgules, les unités, les dizaines... On peut ajouter des 0 si nécessaire.
Exemple : Calculer 8,5 − 2,87

3) Poser une multiplication

Règle : Pour multiplier deux nombres décimaux :
① On pose la multiplication sans tenir compte des virgules
② On effectue la multiplication
③ On place la virgule dans le résultat : le nombre de chiffres après la virgule est égal à la somme des chiffres après la virgule dans chaque facteur
Exemple : Calculer 5,68 × 3,4

II. Division euclidienne

Définition : Diviser un entier (le dividende) par un autre (le diviseur), c'est trouver le quotient et le reste tels que :
dividende = diviseur × quotient + reste
avec le reste strictement inférieur au diviseur
Exemple : Calculer 1347 ÷ 24

III. Vocabulaire

Le résultat d'une addition s'appelle la somme. Les nombres s'appellent les termes.
Le résultat d'une soustraction s'appelle la différence. Les nombres s'appellent aussi les termes.
Le résultat d'une multiplication s'appelle le produit. Les nombres s'appellent les facteurs.
Le résultat d'une division s'appelle le quotient.

IV. Critères de divisibilité

Un entier est divisible par 2 si son chiffre des unités est 0, 2, 4, 6 ou 8.
Un entier est divisible par 3 si la somme de ses chiffres est divisible par 3.
Un entier est divisible par 4 si le nombre formé par ses deux derniers chiffres est divisible par 4.
Un entier est divisible par 5 si son chiffre des unités est 0 ou 5.
Un entier est divisible par 9 si la somme de ses chiffres est divisible par 9.
