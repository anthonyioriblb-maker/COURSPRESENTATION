Chapitre 12 : Les Équations

I. Qu'est-ce qu'une équation ?

Définition : Une équation est une égalité comportant une ou plusieurs inconnues (représentées par une lettre).
On appelle membre de gauche l'expression à gauche du signe =, et membre de droite l'expression à droite du signe =.
Exemples d'équations :
Équation Membre de gauche Membre de droite Degré
3x + 5 = 14 3x + 5 14 1 (équation du 1er degré)
2x − 1 = x + 4 2x − 1 x + 4 1 (équation du 1er degré)
x² + 3 = 7x x² + 3 7x 2 (hors programme de 4e)
⚠ Remarque : En 4e, on étudie uniquement les équations du 1er degré à une inconnue, c'est-à-dire de la forme ax + b = cx + d.
Définition : Résoudre une équation, c'est trouver toutes les valeurs de l'inconnue qui rendent l'égalité vraie.
Une telle valeur s'appelle une solution de l'équation.
Exemple : Considérons l'équation 3x + 5 = 14.
Si on remplace x par 3 :
3 × 3 + 5 = 9 + 5 = 14 ✔
L'égalité est vraie : 3 est solution de l'équation.
Si on remplace x par 2 :
3 × 2 + 5 = 6 + 5 = 11 ≠ 14 ✘
L'égalité est fausse : 2 n'est pas solution de l'équation.

II. Vérifier si un nombre est solution

Méthode : Pour vérifier si un nombre est solution d'une équation, on remplace l'inconnue par ce nombre dans les deux membres séparément, puis on compare les résultats.
Important : Les calculs des deux membres doivent toujours être écrits séparément (ne pas mélanger).
Exemple : L'équation est 2x + 3 = x + 16. Vérifier que x = 13 est solution.

III. Résoudre une équation

1) Les règles de résolution

Propriétés : On peut effectuer les mêmes opérations sur les deux membres d'une équation sans changer les solutions. En particulier :
  • Règle d'addition : on peut ajouter ou soustraire un même nombre des deux membres.
  • Règle de multiplication : on peut multiplier ou diviser les deux membres par un même nombre non nul.
Remarque : Ces propriétés permettent d'obtenir des équations plus simples, ayant les mêmes solutions. On dit que les équations sont équivalentes.

2) Équations du type ax + b = cx + d

Méthode : Pour résoudre une équation du type ax + b = cx + d :
  1. On regroupe les termes en x d'un même côté en soustrayant (ou ajoutant) le terme en x le plus petit.
  1. On regroupe les nombres de l'autre côté.
  1. On résout l'équation obtenue et on vérifie.
Exemple 3 : Résoudre 2x + 3 = x + 16
Étape 0 / 5
Équation de départ
2x + 3 = x + 16   ← on veut supprimer ce x
On soustrait x des deux membres pour faire disparaître le x à droite
2x + 3 − x  =  x + 16 − x
On simplifie, le x de droite a disparu. Maintenant on veut supprimer le +3
x + 3 = 16
On soustrait 3 des deux membres pour faire disparaître le 3 à gauche
x + 3 − 3  =  16 − 3
Le 3 a disparu → solution
Solution : x = 13
Vérification dans l'équation de départ
MG : 2 × 13 + 3 = 26 + 3 = 29
MD : 13 + 16 = 29
MG = MD = 29 ✔
Exemple 4 : Résoudre 5x − 2 = 2x + 7
Étape 0 / 7
Équation de départ
5x − 2 = 2x + 7   ← on veut supprimer ce 2x
On soustrait 2x des deux membres pour faire disparaître le 2x à droite
5x − 2 − 2x  =  2x + 7 − 2x
On simplifie, le 2x de droite a disparu. Maintenant on veut supprimer le −2
3x − 2 = 7
On ajoute 2 des deux membres pour faire disparaître le −2 à gauche
3x − 2 + 2  =  7 + 2
On simplifie, le −2 a disparu. Il reste à isoler x
3x = 9
On divise les deux membres par 3 pour isoler x
3x3  =  93
On simplifie → solution
Solution : x = 3
Vérification dans l'équation de départ
MG : 5 × 3 − 2 = 15 − 2 = 13
MD : 2 × 3 + 7 = 6 + 7 = 13
MG = MD = 13 ✔

IV. Mettre un problème en équation

Méthode en 4 étapes :
1. Choisir l'inconnue : nommer la quantité cherchée par une lettre (souvent x).
2. Traduire le problème en équation : exprimer toutes les données à l'aide de x et écrire l'égalité correspondant à la condition du problème.
3. Résoudre l'équation.
4. Vérifier et conclure en répondant à la question posée dans le contexte.

1) Exemples de mise en équation simple

Exemple 1 (triangle isocèle) :
Les deux côtés égaux d'un triangle isocèle mesurent 2,8 cm. Son périmètre est 10,1 cm. Calculer la longueur de sa base.
Étape 1 – Inconnue :
Soit x la longueur de la base (en cm).
Étape 2 – Équation :
Périmètre = côté + côté + base
2,8 + 2,8 + x = 10,1
soit  5,6 + x = 10,1
Étape 3 – Résolution :
5,6 + x − 5,6 = 10,1 − 5,6
x = 4,5
Étape 4 – Vérification et conclusion :
2,8 + 2,8 + 4,5 = 10,1 ✔
La base du triangle mesure 4,5 cm.
Étape 0 / 4
Exemple 2 (prix au kg) :
3 kg de pommes coûtent 7,50 €. Quel est le prix d'un kilogramme de pommes ?
Inconnue : Soit x le prix d'un kg (en €).
Équation : 3 kg coûtent 3 × prix d'un kg, donc
3 × x = 7,50
Résolution :
3x3  =  7,503
x = 2,50
Conclusion : Un kilogramme de pommes coûte 2,50 €. ✔

2) Problème avec plusieurs inconnues : le choix de l'inconnue

Problème : Léa, Emma et Céline collectionnent les cartes postales. Voici leur conversation :

Emma : « Tu as vu Céline, j'ai deux fois plus de cartes que toi. »

Céline : « C'est vrai mais j'en ai 5 de plus que Léa. »

Léa : « À nous trois, on a 87 cartes. »

Combien de cartes postales possèdent-elles chacune ?
Selon le choix de l'inconnue, l'équation change. Voici les 3 possibilités :
Cas 1 : Soit x le nombre de cartes d'Emma.

EmmaCélineLéa
x x2 x2 − 5
Elles ont 87 cartes à elles trois donc :
x + x2 + x2 − 5 = 87   soit   2x − 5 = 87
2x = 92  →  x = 46

Emma : 46 cartes  |  Céline : 23 cartes  |  Léa : 18 cartes
Vérification : 46 + 23 + 18 = 87 ✔
Cas 2 : Soit x le nombre de cartes de Léa.

EmmaCélineLéa
2 × (x + 5) x + 5 x
Elles ont 87 cartes à elles trois donc :
2 × (x + 5) + (x + 5) + x = 87
2x + 10 + x + 5 + x = 87
4x + 15 = 87
4x = 72  →  x = 18
Léa : 18 cartes  |  Céline : 23 cartes  |  Emma : 46 cartes
Vérification : 46 + 23 + 18 = 87 ✔
Cas 3 : Soit x le nombre de cartes de Céline.

EmmaCélineLéa
2x x x − 5
Elles ont 87 cartes à elles trois donc :
2x + x + (x − 5) = 87
4x − 5 = 87
4x = 92  →  x = 23
Céline : 23 cartes  |  Emma : 46 cartes  |  Léa : 18 cartes
Vérification : 4 × 23 − 5 = 87 ✔
