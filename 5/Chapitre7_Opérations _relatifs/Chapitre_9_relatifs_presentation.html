Chapitre 7 : Opérations avec les relatifs

I. Addition de nombres relatifs

💡 Méthode : Pour additionner deux nombres relatifs, on commence par regarder s'ils ont le même signe ou s'ils sont de signes différents.

I. Addition de nombres relatifs

1) Même signe

Propriété : Pour additionner deux nombres relatifs de même signe :
• On additionne les distances à zéro des deux nombres ;
• On prend le signe des deux nombres.
Exemples :
• ( +2 ) + ( +5 ) = +7
• ( −2 ) + ( −5 ) = −7
• ( +3 ) + ( +9 ) = +12
• ( −8 ) + ( −7 ) = −15

I. Addition de nombres relatifs

2) Signes contraires

Propriété : Pour additionner deux nombres relatifs de signes contraires :
• On prend le signe du nombre qui a la plus grande distance à zéro ;
• On soustrait les distances à zéro.
Exemples :
• ( −2 ) + ( +6 ) = +4
• ( −5 ) + ( +2 ) = −3
• ( +4 ) + ( −11 ) = −7
• ( +8 ) + ( −2 ) = +6

II. Opposé d'un nombre relatif

Définition : L'opposé d'un nombre relatif est le nombre qui a la même distance à zéro mais le signe contraire.
Exemples :
• L'opposé de +3 est −3
• L'opposé de −7 est +7
• L'opposé de +11 est −11
• L'opposé de −5 est +5
💡 Remarque : La somme d'un nombre et de son opposé est toujours égale à 0.
Exemple : ( +3 ) + ( −3 ) = 0

III. Soustraction de nombres relatifs

Propriété : Soustraire un nombre, c'est ajouter son opposé.
Exemples :
• ( +2 ) − ( +3 ) = ( +2 ) + ( −3 ) = −1
• ( +2 ) − ( −3 ) = ( +2 ) + ( +3 ) = +5
• ( −5 ) − ( +8 ) = ( −5 ) + ( −8 ) = −13
• ( −5 ) − ( −8 ) = ( −5 ) + ( +8 ) = +3

IV. Enchaînement d'opérations

Méthode : Pour calculer une expression avec plusieurs additions et soustractions de relatifs :
• Étape 1 : On transforme toutes les soustractions en additions de l'opposé ;
• Étape 2 : On élimine les paires de nombres opposés s'il y en a ;
• Étape 3 : On regroupe les termes positifs ensemble et les termes négatifs ensemble ;
• Étape 4 : On additionne les termes positifs d'un côté, les termes négatifs de l'autre ;
• Étape 5 : On effectue le dernier calcul.
Exemple : Calculer A = ( −6 ) + ( +2 ) − ( −4 ) + ( +5 ) + ( −2 ) − ( +11 )
Étape 1 : A = ( −6 ) + ( +2 ) + ( +4 ) + ( +5 ) + ( −2 ) + ( −11 )
Étape 2 : Éliminer ( +2 ) et ( −2 ) : A = ( −6 ) + ( +4 ) + ( +5 ) + ( −11 )
Étape 3-4 : Regrouper : A = ( +4 ) + ( +5 ) + ( −6 ) + ( −11 ) = ( +9 ) + ( −17 )
Étape 5 : A = −8
