Chapitre 10 : Fonctions affines et linéaires

I. Fonction linéaire

2) Définition et représentation graphique

Définition : Étant donné un nombre a, la fonction linéaire f de coefficient a est la fonction qui à tout nombre x associe le nombre ax.
On note : f : x → ax   ou   f(x) = ax
Exemple : Soit la fonction f qui à tout nombre x associe son quadruple.
On a f(x) = 4x. C'est une fonction linéaire de coefficient 4.
Propriété : Un repère étant choisi, la représentation graphique de la fonction linéaire f : x → ax est une droite passant par l'origine du repère.
Réciproquement, toute droite passant par l'origine (sauf l'axe des ordonnées) est la représentation graphique d'une fonction linéaire.
Le nombre a est appelé le coefficient directeur de la droite.

Exemple : tracer la courbe de f(x) = 3x

Méthode : La courbe est une droite passant par l'origine. Il suffit de calculer un deuxième point.
On choisit le point d'abscisse x = 2 :
f(2) = 3 × 2 = 6
La courbe passe donc par l'origine O(0 ; 0) et par le point A(2 ; 6).
L'équation de la droite est y = 3x. Le coefficient directeur est 3.

II. Fonctions affines

Définition : Soient a et b deux nombres. On appelle fonction affine une fonction qui à tout nombre x associe ax + b.
On note : f : x → ax + b   ou   f(x) = ax + b
Cas particuliers :
— Si b = 0, alors f(x) = ax : c'est une fonction linéaire.
— Si a = 0, alors f(x) = b : c'est une fonction constante.
Propriétés :
— La représentation graphique de la fonction affine f : x → ax + b est une droite.
— Réciproquement, toute droite non parallèle à l'axe des ordonnées est la représentation graphique d'une fonction affine.
— a est le coefficient directeur et b est l'ordonnée à l'origine de la droite.

Exemple : tracer la courbe de f(x) = 2x − 3

Méthode : La courbe est une droite. On calcule les coordonnées de deux points.
On choisit x = 0 et x = −2 :
f(0) = 2 × 0 − 3 = −3   (ordonnée à l'origine)
f(−2) = 2 × (−2) − 3 = −4 − 3 = −7
La courbe passe par A(0 ; −3) et par B(−2 ; −7).

III. Proportionnalité des accroissements

Propriété : Soit une fonction affine f : x → ax + b. Soient x₁ et x₂ deux nombres.
L'accroissement des images est proportionnel à l'accroissement des abscisses, le coefficient étant a.
a = f(x₂) − f(x₁) x₂ − x₁
Cette formule permet de calculer le coefficient directeur à partir de deux points connus.
Cette propriété signifie que quand x augmente de 1, l'image augmente de a. Si a > 0 la droite est croissante ; si a < 0 elle est décroissante.

Exemple : déterminer l'expression de g(x)

Énoncé : Soit une fonction affine g telle que g(−1) = 2 et g(3) = 8. Déterminer l'expression algébrique de g.
La fonction g est affine, donc de la forme g(x) = ax + b.
Étape 1 — Calcul de a :
a = g(−1) − g(3)−1 − 3 = 2 − 8−1 − 3 = −6−4 = 1,5
Donc g(x) = 1,5x + b.
Étape 2 — Calcul de b (méthode 1 : par le calcul) :
On utilise, par exemple, g(3) = 8 :
1,5 × 3 + b = 8
4,5 + b = 8
b = 8 − 4,5
b = 3,5
Ainsi : g(x) = 1,5x + 3,5

Méthode 2 : détermination graphique de b

Méthode graphique : L'ordonnée à l'origine b se lit directement sur le graphique — c'est l'ordonnée du point d'intersection de la droite avec l'axe des ordonnées.
Pour g(x) = 1,5x + 3,5, la droite coupe l'axe des ordonnées au point C(0 ; 3,5).
On lit directement b = 3,5 à l'intersection avec l'axe des ordonnées.   Ainsi : g(x) = 1,5x + 3,5

Récapitulatif du chapitre

Fonction linéaire : f(x) = ax — droite passant par l'origine.
a = coefficient directeur.
Fonction affine : f(x) = ax + b — droite quelconque (non parallèle à l'axe des ordonnées).
a = coefficient directeur  ;  b = ordonnée à l'origine.
Trouver a à partir de deux points connus (x₁ ; y₁) et (x₂ ; y₂) :
a = y₂ − y₁x₂ − x₁
Puis remplacer dans f(x) = ax + b avec un point pour trouver b.
Si a > 0 : droite croissante.    Si a < 0 : droite décroissante.    Si a = 0 : droite horizontale (fonction constante).
