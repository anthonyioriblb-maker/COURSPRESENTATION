Chapitre 12 : Les Équations

I. Qu'est-ce qu'une équation ?

Définition : Une équation est une égalité comportant une ou plusieurs inconnues (représentées par une lettre).
On appelle membre de gauche l'expression à gauche du signe =, et membre de droite l'expression à droite.
Exemples d'équations :
ÉquationMembre de gaucheMembre de droite
3x + 5 = 143x + 514
2x − 1 = x + 42x − 1x + 4
Définition : Résoudre une équation, c'est trouver toutes les valeurs qui rendent l'égalité vraie.
Une telle valeur s'appelle une solution de l'équation.
⚠ Remarque : En 4e, on étudie les équations du 1er degré à une inconnue : ax + b = cx + d.

II. Vérifier si un nombre est solution

Méthode : On remplace l'inconnue par le nombre dans les deux membres séparément, puis on compare.
Important : Les calculs des deux membres doivent être écrits séparément.
Exemple : L'équation est 2x + 3 = x + 16. Vérifier que x = 13 est solution.
Membre de gaucheMembre de droite
2 × 13 + 3 = 26 + 3 = 2913 + 16 = 29
Les deux membres sont égaux → x = 13 est bien solution ✔

III. Résoudre une équation

Les règles de résolution

Propriétés : On peut effectuer les mêmes opérations sur les deux membres sans changer les solutions :
• Règle d'addition : on peut ajouter ou soustraire un même nombre des deux membres.
• Règle de multiplication : on peut multiplier ou diviser les deux membres par un même nombre non nul.
Remarque : Ces opérations produisent des équations équivalentes (mêmes solutions).
Méthode pour résoudre ax + b = cx + d :
1. Regrouper les termes en x d'un même côté.
2. Regrouper les nombres de l'autre côté.
3. Résoudre et vérifier.

Exemples de résolution

Exemple 1 : Résoudre 2x + 3 = x + 16
2x + 3 = x + 16
2x + 3 − x = x + 16 − x
x + 3 = 16
x + 3 − 3 = 16 − 3
x = 13
Vérification : MG = 2×13+3 = 29 ; MD = 13+16 = 29 ✔
Exemple 2 : Résoudre 5x − 2 = 2x + 7
5x − 2 − 2x = 2x + 7 − 2x
3x − 2 + 2 = 7 + 2
3x = 9 → x = 3
Vérification : MG = 15−2 = 13 ; MD = 6+7 = 13 ✔

Équations avec parenthèses et fractions

Méthode : Développer les parenthèses (distributivité) avant de regrouper.
Exemple : Résoudre 3(x + 2) = 2x − 1
3x + 6 = 2x − 1
3x − 2x = −1 − 6
x = −7
Vérification : MG = 3(−7+2) = 3×(−5) = −15 ; MD = 2×(−7)−1 = −15 ✔
Méthode (fractions) : Multiplier tous les membres par le dénominateur pour éliminer les fractions.
Exemple : Résoudre x + 13 = 2x − 16
× 6 :   2(x + 1) = 2x − 1
2x + 2 = 2x − 1    →    2 = −1 : impossible → pas de solution

Problèmes du quotidien

Méthode : Pour résoudre un problème avec une équation :
1. Choisir l'inconnue et la nommer (x = ...)
2. Traduire la situation en équation
3. Résoudre l'équation
4. Rédiger la réponse
Exemple : Le double d'un nombre diminué de 5 est égal au triple de ce nombre augmenté de 2. Trouver ce nombre.
Soit x le nombre cherché.
2x − 5 = 3x + 2
−5 − 2 = 3x − 2x
x = −7
Le nombre cherché est −7.
